About

Welcome to Lin!
This is a linear equation solver app. How to use? Quite easy. On loading this page, two equation fields have been provided for you already. Two fields are the minimum number of fields you can have. This app is capable of solving a large number of system linear equations (although it'll obviously take more time for larger inputs). Once you're done filling the fields, click '=' on the footer to display the results.

NOTE:

  • Empty fields are not tolerated. Please remove all empty fields if this is to work smoothly.
  • Does not automatically simplify any of the expressions nor does it collect 'like terms'. As of now please do that yourself.

Please report bugs to francisali692@gmail.com

Well then go on ahead and solve!
